Card Usage Guidelines

Core Program Cards: Use for main features/benefits with high value. Large icons (w-16 h-16), bold titles (1.44rem), prominent value badges.

Partner Perks Cards: Use for secondary features/benefits. Smaller icons (w-12 h-12), medium titles (1.2rem), compact value badges.

Fast-Action Bonuses Cards: Use for limited-time offers or special bonuses. Red color scheme to create urgency, same layout as Core Program cards.

Common Elements: All cards use rounded-xl corners, hover effects, transition animations, and consistent spacing (p-6).
