Buttons

Primary Button

Style: Red background (#dc2626), white text, Sora font

Size: Large padding (32px x 16px), 18px font size

Effects: Shadow, subtle lift on hover

CSS Class: bg-primary text-white px-8 py-4 rounded-lg

Secondary Button

Style: Transparent background, red border, red text

Size: Same padding as primary button

Effects: Hover fills with red background

CSS Class: border-2 border-primary text-primary hover:bg-primary

Ghost Button

Style: Transparent background, white border, white text

Size: Same padding as primary button

Effects: Hover changes text and border to accent color

CSS Class: border-2 border-white text-white hover:text-accent
